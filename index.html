使用说明

这个版本默认使用本地生成器，适合你马上测试交互和文案体验。 如果后续要接真实大模型，只需要把右侧“提示词预览”接到你的 API 即可。
支持关键词联想、结构化输出、复制、重写和本地测试。你也可以后续把它接成真正的“输入词语 -> 大模型生成 rap 歌词”的线上智能体。
